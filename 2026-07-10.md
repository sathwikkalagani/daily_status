## What I Learned

* Structures allocate separate memory for each member.
* Unions share the same memory among all members.
* `typedef` creates an alias for an existing data type.
* Alignment is a memory placement rule; padding is extra memory added to satisfy that rule.
* `#pragma pack(1)` reduces or removes padding.
* Enum values are integers and increase automatically unless explicitly assigned.
